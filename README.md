# Data Engineering Pipeline - Repo Skeleton

## Overview
This project contains a production-style data engineering pipeline:
- Ingestion -> Staging -> Transform -> Curated
- Orchestration: Airflow / ADF
- Storage: S3 / ADLS
- CI/CD: GitHub Actions

## Setup
1. python -m venv .venv
2. source .venv/bin/activate
3. pip install -r requirements.txt
